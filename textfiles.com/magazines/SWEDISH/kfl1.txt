1. Skaffa en wingate (så du kan telnetta med wingaten's ipnr.)
2. Fixa nu ett anonymt shell (typ hackat), telnetta bara via wingaten

4. Leta reda på en http Proxy server (tips sök på host:proxy i altavista)
5. Mappa en lokal port på shellburken till http Proxy porten på proxyn (80 oftast)
   typ "./datapipe 8818 8080 proxy.domän.."
6. Configga browsern med shellhost port 8818 som Proxyserver.


Dom som driver sajten där du cardat kan få fram är proxyns IP och shellets IP,
sen kontaktar dom shellhostern som bara ser wingatens IP i connections från
