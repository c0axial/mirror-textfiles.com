 * 3 * Hur man gör en spänningsmultiplikator ********| Electrodeath |*
 *********************************************************************
 
 Hejsan medmänniskor!
 Idag  tänkte jag lära ut hur man bygger en enkel stungun. Jag hittade
 en  beskrivning i The Lib och fann att man utelämnat en hel del info.
 Som namnet visar så är det en grunka som multiplicerar spänningen.
 Ett steg är en diod och en kondensator, så här:
 
 ------+   Detta  är  ETT  steg  som dubblar EN gång men notera att de
       |   första  två stegen räknas som ett steg.Det finns inte någon
       |   gräns  för hur många steg du kan bygga på men vid en 12000-
       |   15000 volt  kan det bli lite svårt med isolationen eftersom
       A   de  små elektronerna vill hoppa. Vid 230 vac ansluter du en
       |   stickpropp!  När  du laddat upp  den (tar 5 sek) tänk då på
       |   att  stickproppen är lika mycket strömförande som resten av
       |<-- diod
       |   maskinen  dvs. du får strömmen i DIG istället för offret om
 --||--+   du petar på stickproppen.
 kondensator
 
 
    Här är den som jag byggde med 4 steg alltså 3 x 230= 690 v
 
    +------------+-||-+----+-||-+
    *            |    |    |    |
                 |    |    |    |
   230VAC        A    V    A    V
                 |    |    |    |
    *            |    |    |    |
    +----+----||-+----+-||-+----+-+
         |                        |
         |                        |
         |                        |
         |                        |
         -------- 690v ut ---------
 
 A= dioden har minus uppåt   V=dioden har + uppåt
 
 Du  kan löda komponenterna på en bräda med spikar som kontakter eller
 kretskort (rekommenderas).  Lämplig  inneslutning  är  en metall/trä/
 plastlåda av  något  slag. Om du exv. har 30 steg så kan du innesluta
 den i ett långt hölje och kan på tryggt avstånd zappa fienden/vännen.
 Kondensatorerna skall vara av typen X-2 kapacitans: 0.1 uF
 Dioderna skall vara av typen 1N4007 och den är på: 1 A 1000 volt
 elfanummer för kondensatorn: 65-646-86 10:-
 elfanummer för dioden: 70-005-16 2:-
 elfanummer för resistorn: 60-788-10 4:- 
 Den  tål inte kortslutning men om du kopplar in en 470 kilo-ohms res-
 istor i serie med  apparaten så kortsluts den aldrig helt, se bild:
 
 
    +------------+-||-+----+-||-+
    *            |    |    |    |
                 |    |    |    |
   230VAC        A    V    A    V
                 |    |    |    |
    *            |    |    |    |   __
    +----+----||-+----+-||-+----+--|__]-+
         |                              |
         |                              |
         |                              |
         |                              |
         ------------- 690 v-------------
 
 Förresten; Apparaten likriktar spänningen också!
 Litet lexikon för icke-elektriker:
 Volt (v) = Spänning
 Ampere (A) = Ström
 Ohm = Resistans (motstånd mot ström)
 Kyss = Stark stöt [/Mr. H]
 
 Om någon vill ha hjälp eller mer info så maila!
 
 PS. Om nån vill ha en kretskortsförlaga på denna hör av er!
 
 
 ELECTRODEATH

 * 4 * The Ultimate Djurplågeriguide ****************| Electrodeath |*
 *********************************************************************
 
 Efter att många stackars förtappade själar skrivit en massa patetiska
 dödshot  till mig så kom jag på att tänka på en grej; om 10 djurplåg-
 eritips kan provocera så många människor så måste man ju göra om det!
 Som  vanligt är det mig ni ska skicka hotbrev till och inte Mr. Head-
 ache. [Precis. Och skicka många. /Mr. H]
 
 ! Här är flera tips:
 
 1.Man  kan  sätta  allt från batterier till 25 kv transformatorer på
 nosen på hunden, katten, hamstern etc etc
 
 2.Bind hunden och sätt upp den i ett träd och öva skytte eller kasta
 dartpil. Garanterade ljudeffekter vid träff hehe...
 
 3.Ta en visselpipa (helst ultraljudspipa) och blås allt vad du kan i
 hundens öra.
 
 4.Släng  in en chinchilla eller hamster i mikrovågsugnen och sätt på
 defrost och se hur länge den lever!
 
 5.Spraya  djuret  i  någon kul färg t.ex lilametallic (cred till Mad
 Alien!) 
 
 6.Klassikern  med  dijonsenap funkar lika bra med vanlig senap. Sätt
 en klick i hundens röv and watch him run!
 
 7.Om  du  ser en Dogo Argentino,pitbull eller en (2!) rottweilers så
 måste  du  bara  sparka allt vad du kan. (Credits till Twgrr som har
 skrivit flest hotbrev varav ett ni kan läsa nedan)
 
 8.Ta  smällare eller hemmagjorda bomber och stoppa i röven/munnen på
 djuret. BOOM >:]
 
 9.Släng  i nån sorts sprit i ett akvarium (i USA gjorde en filur det
 i en delfinbassäng! Han hällde i 50 l rödsprit!) och se hur fiskarna
 simmar uppochner =)
 
 10.Skär  upp  ditt  lilla  husdjur  och titta inuti (gärna medan det
 lever).
 
 Här är det bästa erbjudandet jag nånsin fått:
 
 Få  se  om  du är så jävla stor i käften när du möter mig. Maila mig
 så vi  kan bestämma en tid och plats (helst i en skog, där ingen kan
 stoppa  oss)  att  mötas på, jag är helt seriös. Du får ta med vilka
 vapen  som  helst  och  hur många kamrater du kan. Var inte så jävla
 feg,  kom  och möt mig.du vågar väl stå för dina åsikter? Jag kommer
 att  ha  med  mig  mina hundar som du kan testa att plåga, två rott-
 weilers, en pitbull och en dogo argentino. Visa att du står för dina
 åsikter, eller gömmer du dig bakom en bildskärm och leker huligan.
 Visa vad du står för.
 
 //Twgrr

 * 5 * Sluta snusa *****************************************| Knash |*
 *********************************************************************
 
 Tjo...kanske kommer lite sent men.. jag har ju också börjat snusa :-)
 Tips för att sluta snusa:
 
 Avbryt snusningen tvärt.
 Kasta  bort alla snusdosor och annat som har med snusningen att göra.
 Övergå inte till att röka eller att blanda rökning och snusning.
 Ät  färsk  frukt  eller  sockerfritt  tuggummi  om du behöver något i
 munnen.
 Smaker som pepparmint, lakrits ochapelsin brukar dämpa begäret.
 Tala om för din omgivning att du slutatsnusa och att du inte vill bli
 bjuden på snus.
 Var  beredd  på  att  du blir irriterad och får koncentrationssvårig-
 heter. Det går över.
 Lär  dig  att  slappna av när du känner suget efter nikotin. Apoteket
 säljer kassettband med avspänningsträning.
 Undvik  miljöer där du träffar andra snusare den närmaste tiden efter
 ditt stopp.
 Drick mycket vatten.
 Borsta tänderna så snart du blir snussugen.
 Gå till tandläkaren och ta bort tandsten och nikotinbeläggning, samt
 kontrollera tandköttet.
 
 Fördelar när du har slutat:
 Du känner dig ren och fräsch i munnen.
 Du luktar inte snus.
 Maten smakar bättre.
 Bättre ekonomi.
 Risken för tandlossning minskar.
 Du slipper vara beroende av en drog.
 Händer och fötter håller sig varmare.

 Goddagens  gott  folk.  Kom  på  en sak när jag satt och hexeditade i
 filen  mIRC32.exe,  med  andra  ord IRC-klienten mIRC, som för övrigt
 suger ganska  mycket. Well, den suger faktiskt så mycket att man inte
 vill visa att man har den.. det är typ lamhetsförklarning på program-
 fanskapet.  Men  om  man  använder Windows (*urk*) så har man inte så
 stort  val..  de  andra klienterna suger minst lika mycket de.. Okej,
 okej  det  FINNS undantagsfall.. men det är oväsentligt just nu.. vad
 jag  ville  få ut av denna artikeln var hur man hexeditar mIRC så att
 allt  tufft  folk med BitchX och UN*X inte kan avslöja din lamhet att
 sitta i just mIRC..
 Okej,  först  och främst laddar du ner en hexeditor.. jag förutsätter
 att  du vet vad det är för något. Jag kan starkt rekommendera Hexedit
 av  JBA  Software,  en  simpel  men  bra editor för Windows utan alla
 psychofunktioner  som  omger  alla  program  numera (så vida man inte
 dumpar Windows, vilket du, ni, jag, dem och ALLA borde göra..).
