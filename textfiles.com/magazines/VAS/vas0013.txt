        Back again for more interesting info. This issue is going to explain
to you a bit about the more commonly known boxes, It will tell you such
interesting information as who devised it, and what exactly it does... So
without any further adeu...

Box Name : Acrylic Box
Creator  : The Pimp
Used For : To Get Three-Way-Calling, Call Waiting, Programmable Call
           Forwarding, and an easier way of extended Bud Boxing All for
           Free.

Box Name : All-In-One Box
Creator  : Disk Doctor
Used For : A basic switching system, Line-Priority for your house.

Box Name : Aqua Box
Creator  : Unknown
Used For : To escape the F.B.I. "Lock In Trace" method of tracing calls.

Box Name : Assasin Box
Creator  : Grim Reaper
Used For : To cause damage to the person or computer on the other end
           of the phone via electrical shock.

Box Name : The Beagan Box
Creator  : Black Box
Used For : Tappin into other peoples home phones and using them from your
           house to call long distance.

Box Name : The Beige Box
Creator  : The Exterminator And The Terminal Man
Used For : This is a replacement for a linesman's handset, very useful
           device.

Box Name : The Black Box
Creator  : Unknown
Used For : When people call you phone number long distance, they won't be
           charged for the call.

Box Name : The Blast Box
Creator  : Shadow Demon
Used For : To create a very loud noise via amplifier over the phone line to
           destroy the other person's eardrums.

Box Name : The Bloto Box
Creator  : Aftermath
Used For : This box will cause electrical interference with the phone system
           causing all phones to ring, and people picking up their fones
           near the box will get electrocuted. Very powerful.

Box Name : The Blue Box
Creator  : Unknown
Used For : To gain access to the trunk and make free phone calls all over
           the world.

Box Name : The Bob Box
Creator  : Unknown
Used For : This is similar to a beige box (See above), only it has plans
           For a wireless transmitter/receiver.

Box Name : Brown Box
Creator  : Unknown
Used For : This is used to create a party line out of 2 lines at your house.

Box Name : Bud Box
Creator  : Mad Hacker
Used For : This is basically just a portable version of our friend, the
           beige box.

Box Name : Cheese Box
Creator  : Unknown
Used For : Turning your phone into an untraceable payphone. This requires a
           red box to work properly.

Box Name : Clear Box
Creator  : The Bit
Used For : This box is used to make free fone calls on post-pay public phones.

Box Name : Color Box
Creator  : Aluminum Gerbil
Used For : This box is basically used to tap and record phone conversation
           and the such.

Box Name : The Copper Box
Creator  : The Cypher
Used For : This box can be used to destroy a phone system. It does so by
           overloading the circuts on the phone lines, possibly causing
           fires and such at the buildings.

Box Name : Crimson Box
Creator  : Unknown
Used For : This box is used to either make your phone busy, or put someone
           currently on the line on hold.

Box Name : Day-Glo Box
Creator  : John F. Kennedy
Used For : This is used to make phone calls for free with no time limit.
           Basically a better-put-together beige box.

Box Name : DLOC Box
Creator  : The Lords Of Chaos
Used For : This is used for merging 2 lines together into a conferencing type
           of chat.

Box Name : Gold Box
Creator  : Dr. Revenge
Used For : This box is used to make your line into an untracable second
           phone line.

Box Name : Green Box
Creator  : The Blue Buccaneer
Used For : This is basically a red box that you can't use on a phortress
           phone <gee, how useful>.

Box Name : Jack Box
Creator  : Unknown
Used For : This is used to modify the tones on your telephone touchpad.

Box Name : Lunch Box
Creator  : Dr. De-Code
Used For : A Portable phone tapper. It has a small receiver to help keep it
           less obvious.

Box Name : Mauve Box
Creator  : Captain Generic
Used For : This box is used to pull in the nearest phone conversation around
           you. Simply plug your phone in, and listen.

Box Name : Neon Box
Creator  : Unknown
Used For : Used to record things off the phone, or allows a clear passage for
           your computer-generated tones to be sent out over the modem.

Box Name : Noise Box
Creator  : Unknown
Used For : This is basically a line-noise filter. Somewhat useless in the
           age of the MNP or v.42bis HST, but still usefull for you guys
           still at 2400.

Box Name : Party Box
Creator  : Greyhawke
Used For : This is used for making 3-way calls and such without actually
           having 3 way calling. Similar to a DLOC or Brown box.

Box Name : Pearl Box
Creator  : Dr. De-Code
Used For : This box can replace the red & green box, making tones from
           1-9999mhz.

Box Name : Pink Box
Creator  : Baba O'Riley
Used For : This box is used to port music or anything else directly onto
           your phone line.

Box Name : Purple Box
Creator  : The Flash
Used For : This box simply gives your phone a hold button, with a LED
           indicator to let you know they are on.

Box Name : Razz Box
Creator  : The Razz
Used For : This box is used to tap into your neighbors phone line and call
           all over on it. <What an original concept we have here>.

Box Name : Red Box
Creator  : Unknown
Used For : This box is used to simulate the tones that coins make when
           placed into a payphone.

Box Name : Rock Box
Creator  : Video Vindicator
Used For : This is used to port music from your stereo directly onto your
           phone line, a bit more advanced than the pink box.

Box Name : Scarlet Box
Creator  : The Pimp
Used For : To create a bad connection on the line permenantly. Good to use
           on a lamer's BBS line.

Box Name : Silver Box
Creator  : Unknown
Used For : Used to switch your keypad on the phone between normal and
           1633hz tones.

Box Name : Static Box
Creator  : The Ursurper
Used For : This box is used to regulate the voltage on a line to make sure
           that the line remains static free. Useful for when you are on an
           extender that gives lots of noise.

Box Name : Switch Box
Creator  : Unknown
Used For : This box is used for basic control of your home phone lines.

Box Name : Sword Box
Creator  : Grim Reaper
Used For : This is basically a Beige/Bud box with some minor enhancements
           to improve performance.

Box Name : Transmitter Box
Creator  : Unknown
Used For : This box is used to generate tones to locate the origin of a
           phone tap.

Box Name : Tron Box
Creator  : Cap'n Crunch
Used For : This is used to slow down your electric meters, and sometimes
           even to put them in reverse.

Box Name : Trunk Box
Creator  : The Data Dragon
Used For : This box is used to tap into the big MA bell boxes around your
           neighborhood.

Box Name : Urine Box
Creator  : Wolfgang von Albatross
Used For : This box is used to play nasty tricks on the person on the other
           end of the phone <i.e. melting his handset, disinegrating him?>.

Box Name : White Box
Creator  : Unknown
Used For : This box is basically a portable handset for a telephone.
