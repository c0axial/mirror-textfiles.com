1. You must first find a ASPEN Voice Mail System phone number. The best ones
   are A) newer ones [more availible mail boxes] B) ones on 1-800 #'s [worth
   more]

2. Once you have found it, call it up using a touch tone phone [if you use
   pulse you are mighty lame]. When you are online you will hear a digitized
   woman giving intructions, or a real person if the system administrator chose
   to customize the greeting. Enter either a 3 digit [form of XXX] or a
   4 digit [form of XXXX] number, do not include any *'s or #'s. If it went
   through, you'll hear a mail box holder say his greeting, then a beep. If
   not you either dialed an invalid mailbox, OR its the other digit format.
   So try again until you get someone saying "Hi this is Ted Bundy...<blah>
   <blah> <blah>, hack hack, scream scream, leave a message". Once you have
   gotten that take the number and add 100 to it [ie. if Ted's box was 387,
   just remember 487, if it was 3887 just remember 3987]. And if it is above
   899 then subtract 200. This is just to make your hacking easier because
   most administrators want the system organized and won't have random
   mailboxes all around the number scale [ie. they won't do it sequential,
   BUT they might keep the numbers between 387 and 487, this is not
   guaranteed, but it might make it a little easier]

3. Now take that number that you were supposed to remember, [or if you
   choose use a random one in the number scale for that system, either
   000-999 or 0000-9999] and get ready to call back. Once you hear that
   initial greeting again, enter the pound or # key and then enter that
   number (say 123) that should be in yer memory, now there are three
   possibilities on what happens next:
        1) 'Hi this is Ted...' <--- No good
        2) '123 is not a valid mailbox...' <--- No good
        3) 'Box #123' <--- Yipee! Horah! Jump fer joy!

   Now, if you got #1, then try again using 124, do not randomize.
   If you got #2, do the same as above.
   If you got #3 you are in luck. You have just accessed a mailbox that
   has not been used yet, call it a virgin mailbox. (just don't try to
   screw it)

4. Ok, now you got yer virgin mailbox, now you must enter a password.
   Some default passwords for ASPEN are:
                1.  1+VMB#   ..........  1123
                2.  VMB#+1   ..........  124
                3.  0+VMB#   ..........  0123
                4.  VMB#+0   ..........  1230
                5.  VMB#     ..........  123
                6.  VMB#+100 ..........  223   Taken from Octel ASPEN Man.
                7.  VMB#-100 ..........  023

5. Try all of them, and remember the system will only have ONE default
   password, so if you get one virgin mailbox, and get the password,
   you'll also be able to access all other virgin mailboxes.

6. If none of them work, then try combinations of VMB +/- xxx. This means
   the System Administrator changed the default, but usually they don't.

7. Once you are in, you can do many things, here is the Octel Comm.
   "Quick Reference Chart" for the ASPEN System (sorry for the crude
   drawings) editors notes will be in parenthesis:

  "TO ACCESS YOUR MAILBOX:
   -----------------------

   1. Call your Octel Communications Voice Processing System.

          If you hear your name, skip to step 5. (direct dials)

   2. Listen to the introductory prompt.

   3. Press [#]

   4. Enter your mailbox number

   5. Enter your password and press [#]

   6. Listen for main menu prompts

The [*] Key

Press [*] whenever you wish to:

          Cancel what you are doing
          Back up to a previous menu
          Erase and re-record
          Re-enter destination number if you make a mistake
          Disconnect when at the Main Menu; if you just hang up,
           you will miss the 'check new messages' prompt

The [#] (pound) Key

Press [#] whenever you wish to:

          Complete a step
          Skip a message you are reviewing; skipped messages
           remain in the queue as new messages
          Send a message
          Skip personal greetings and introductory prompts

Press [#] [#] whenever you wish to:
